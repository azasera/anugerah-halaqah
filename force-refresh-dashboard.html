🔄 Force Refresh Dashboard

Tool untuk memaksa dashboard reload data terbaru dari Supabase

ℹ️ Kapan menggunakan tool ini?

Gunakan tool ini jika data di dashboard tidak update setelah sync, meskipun data di Supabase sudah benar.

✅ Refresh berhasil!

Dashboard akan reload dalam 2 detik...

Cara Manual:

  1. Buka dashboard
  2. Tekan Ctrl + Shift + R (Windows) atau Cmd + Shift + R (Mac)
  3. Atau buka Developer Tools (F12) → Network tab → centang "Disable cache" → refresh
